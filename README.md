# Zig app server

Made for fun but can be a replacement for 'Rotmg Pservers' app server

### Status : WIP

### Requirements
Database: Redis 6.0+ <br/>
Zig version 0.11.0

### Features
No dependencies! <br/>
Redis parser for Zig - Modified version of [ok redis](https://github.com/kristoff-it/zig-okredis) for Zig
